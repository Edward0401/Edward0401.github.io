每次试除时其实只要除到 sqrt(n) 只需要在2到 之间寻找 n 的因数

并且除2之外不需要试除偶数

2.6n ± 1【可以排除掉一大部分】

如果 n 是一个质数（不为2或3）， 那么：

   n = 6n ± 1

因为如果 n 是一个质数（不为2或3）那么 n % 2 != 0 且 n % 3 != 0
