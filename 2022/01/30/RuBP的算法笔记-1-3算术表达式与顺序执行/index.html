RuBP的算法笔记-1.3算术表达式与顺序执行

1.3算术表达式与顺序执行

1. c=double(a)/b;

​ 注意：运算符的结果精度取决于参与运算的数据类型，在运算时发生自动类型转换；先计算再赋值；因此，将结果赋予一个高精度数并不能改变运算的精度。

2.取整方法

C/C++提供了库函数用来向上或向下取整，ceil()与floor().

#include

double floor (double x) // 对x进行向下取整

double ceil (double y) // 对y进行向上取整

另外注意 / 并不是进行四舍五入（7/2 = 3），所以我们可以用

(int)(a/b) +1
来表示向上取整，同样的用

(int)(a/b) -1

来表示向下取整。

而C/C++中存在进行四舍五入运算的库函数RoundEx(double x):

RoundEx(1.4) = 1

RoundEx(1.6) = 2

RoundEx(-1.4) = -1

RoundEx(-1.6) = -2
